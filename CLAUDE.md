# 数独（ナンプレ）アプリ

## アクセス方法

- **URL**: `https://tattu-cat.github.io/sudoku/`
- スマホのSafariで開き、「ホーム画面に追加」するとPWAとして使用可能
- サーバー不要・オフラインでも動作

## ファイル構成

- **`index.html`** — ゲーム本体。パズル生成・検証・UIをすべてクライアントサイドJSで完結。
- **`sw.js`** — Service Worker。オフラインキャッシュを管理する。
- **`manifest.json`** — PWA設定。ホーム画面追加・スタンドアロン起動に対応。

## アーキテクチャ

サーバーレス構成。バックエンド不要。

| 項目 | 内容 |
|------|------|
| ホスティング | GitHub Pages（HTTPS） |
| リポジトリ | `https://github.com/tattu-cat/sudoku` |
| パズル生成 | クライアントサイドJS（バックトラック法、唯一解保証） |
| オフライン | Service Workerによるキャッシュ |

## ゲーム機能

### 難易度
| 難易度 | 除去マス数 |
|--------|-----------|
| かんたん | 36 |
| ふつう   | 46 |
| むずかしい | 52 |
| エキスパート | 56 |

### 実装済み機能
- パズル生成（バックトラック法・MRVヒューリスティック、唯一解保証）
- メモ機能（候補数字を小さく表示、ON/OFFトグル）
- タイマー（新しい問題で自動リセット）
- チェックボタン（誤りマスを2秒間赤くハイライト）
- 正解を見る（確認ダイアログあり）
- クリア検出（全マス正解でアニメーション＋タイム表示）
- ハイライト（同行・同列・同ブロック、同じ数字）
- キーボード操作（矢印キーで移動、数字キーで入力）
- ダークモード対応

## スマホ向け設計方針

- 画面全体にボードが収まるよう CSS で `vmin` / `svh` 単位を使用
- タップ操作: マスをタップして選択 → 数字ボタンをタップして入力
- ピンチズームを無効化（`viewport` の `user-scalable=no`）
- ダークモード対応（`prefers-color-scheme`）
- `input` 要素を使わずカスタム数字ボタンで入力（ソフトキーボード回避）
