# 数独（ナンプレ）アプリ

## 起動方法

```bash
# 依存関係のインストール
pip install -r requirements.txt

# サーバー起動（通常はデスクトップの .command ファイルをダブルクリック）
uvicorn api:app --host 0.0.0.0 --port 8001
```

デスクトップの **「数独を起動.command」** をダブルクリックするとサーバーが自動起動し、ブラウザが開く。
スマホからは `http://<MacのIP>:8001` にアクセス。

## ファイル構成

- **`api.py`** — FastAPI サーバー。パズル生成・検証のAPIと、スマホ向けゲームUIのHTMLを配信する。
- **`sudoku_generator.py`** — 数独パズルの生成ロジック。難易度別に空白マスの数を調整する。
- **`数独を起動.command`** — ダブルクリック起動スクリプト。
- **`requirements.txt`** — 依存ライブラリ一覧。

## アーキテクチャ

| サービス | ポート | 用途 |
|---------|--------|------|
| FastAPI | 8001 | パズルAPI＋スマホ向けゲームUI（HTML配信） |

household-budget（ポート8000）と共存できるよう 8001 を使用。

## 画面構成

```
/ (GET)          → ゲームUI（HTML）
/new  (GET)      → 新しいパズル生成 ?difficulty=easy|medium|hard|expert
/check (POST)    → 回答の検証
/solve (POST)    → 正解を返す（ギブアップ用）
```

## ゲーム機能

### 難易度
| 難易度 | 空白マス数の目安 |
|--------|----------------|
| easy   | 36以下         |
| medium | 37〜45         |
| hard   | 46〜52         |
| expert | 53以上         |

### メモ機能
- 各マスに複数の候補数字（1〜9）を小さく表示できる
- メモモードのON/OFFをボタンで切り替え
- 数字を確定入力するとそのマスのメモはクリア

## スマホ向け設計方針

- 画面全体にボードが収まるよう CSS で `vmin` 単位を使用
- タップ操作: マスをタップして選択 → 数字ボタンをタップして入力
- ピンチズームを無効化（`viewport` の `user-scalable=no`）
- ダークモード対応（`prefers-color-scheme`）
