# 数独（ナンプレ）アプリ

スマホで遊べる数独パズルゲームです。サーバー不要・オフラインでも動作します。

## アクセス

**https://tattu-cat.github.io/sudoku/**

iPhoneのSafariで開き、「ホーム画面に追加」するとアプリのように使えます。

## ファイル構成

```
sudoku/
├── index.html      ← ゲーム本体（HTML + CSS + JS 完結）
├── sw.js           ← Service Worker（オフラインキャッシュ）
└── manifest.json   ← PWA設定
```

## アーキテクチャ

サーバーレス構成。すべてクライアントサイドJSで完結しています。

| 項目 | 内容 |
|------|------|
| ホスティング | GitHub Pages（HTTPS） |
| パズル生成 | クライアントサイドJS（バックトラック法、唯一解保証） |
| オフライン | Service Workerによるキャッシュ |

## ゲーム機能

### 難易度
| 難易度 | 除去マス数 |
|--------|-----------|
| かんたん | 36 |
| ふつう   | 46 |
| むずかしい | 52 |
| エキスパート | 56 |

### 実装済み機能
- パズル生成（バックトラック法・MRVヒューリスティック、唯一解保証）
- メモ機能（入力ボタン下に数字別メモボタン行、選択セルのメモ状態をハイライト表示）
- タイマー（新しい問題で自動リセット）
- 誤入力の即時赤色表示（解答と異なる数字を入力した瞬間に赤色）
- 数字残数カウンター（numpadに残り入力可能数を表示、9個揃ったらグレーアウト）
- チェックボタン（間違い数をメッセージ表示）
- 正解を見る（ギブアップ・確認ダイアログあり）
- クリア検出（アニメーション＋タイム表示）
- ハイライト（同行・同列・同ブロック・同じ数字）
- キーボード操作（矢印キーで移動、数字キーで入力）
- ダークモード対応
- PWA対応（ホーム画面追加・オフライン動作）
